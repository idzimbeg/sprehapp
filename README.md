# Sprehapp

Example todo application developed during Frontend developer course in ALgebra.

## Instructions
1. `npm install`
2. `npm run start`

## Features
- [x] Create Context
- [x] Create login form
- [x] Create message form
- [x] Add date and time
- [x] Style send/received messages
- [X] Create server
- [X] Assign port
- [X] Add Socket.io
- [X] Add CSS styling
